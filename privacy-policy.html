Privacy Policy

PK Chrome Shortcuts is a Chrome extension that provides keyboard shortcuts and tab/window utilities inside Google Chrome.

Data collection

PK Chrome Shortcuts does not collect, store, sell, or share personal data with the developer or third parties.

The extension does not use analytics, advertising trackers, remote logging, or external databases.

How the extension works

The extension uses Chrome extension APIs locally on your device to:

  • manage tabs and windows
  • open Chrome internal pages such as bookmarks, history, downloads, settings, and shortcuts
  • react to keyboard shortcuts and optional click gestures
  • display a badge with the current number of open tabs

Website data and page access

The extension runs on webpages because it provides shortcut and navigation features that depend on the current tab and user interaction on links.

Page access is used locally on your device only. The extension does not transmit webpage contents, browsing history, form inputs, or personal information to any external server.

Permissions

The extension uses the following Chrome permissions:

  • tabs
  • windows
  • activeTab
  • scripting
  • host access on pages required by the extension features

These permissions are used only to provide the extension's local functionality.

Data sharing

No personal data is sold, transferred, or shared with third parties.

Contact

If you have questions about this privacy policy, contact the developer through the support or contact details associated with the extension listing.

Politique de confidentialite

PK Chrome Shortcuts est une extension Chrome qui fournit des raccourcis clavier et des utilitaires de gestion des onglets et fenetres dans Google Chrome.

Collecte de donnees

PK Chrome Shortcuts ne collecte, ne stocke, ne vend et ne partage aucune donnee personnelle avec le developpeur ou des tiers.

L'extension n'utilise ni analytics, ni traceurs publicitaires, ni journalisation distante, ni base de donnees externe.

Fonctionnement de l'extension

L'extension utilise localement sur votre appareil les API Chrome pour :

  • gerer les onglets et les fenetres
  • ouvrir des pages internes de Chrome comme les favoris, l'historique, les telechargements, les parametres et les raccourcis
  • reagir aux raccourcis clavier et aux gestes de clic optionnels
  • afficher un badge avec le nombre actuel d'onglets ouverts

Acces aux pages web

L'extension fonctionne sur les pages web car elle fournit des fonctions de raccourcis et de navigation qui dependent de l'onglet courant et des interactions de l'utilisateur sur les liens.

L'acces aux pages est utilise localement sur votre appareil uniquement. L'extension ne transmet pas le contenu des pages, l'historique de navigation, les champs de formulaire ou des informations personnelles a un serveur externe.

Autorisations

L'extension utilise les autorisations Chrome suivantes :

  • tabs
  • windows
  • activeTab
  • scripting
  • l'acces aux pages necessaires aux fonctionnalites de l'extension

Ces autorisations sont utilisees uniquement pour fournir les fonctionnalites locales de l'extension.

Partage des donnees

Aucune donnee personnelle n'est vendue, transferee ou partagee avec des tiers.

Contact

Si vous avez des questions sur cette politique de confidentialite, contactez le developpeur via les informations de support ou de contact associees a la fiche de l'extension.
